stat 命令查看文件详细信息

在Terminal的命令行中操作时，常需要查看文件的详细信息，包括文件大小、最后的修改时间等。尽管有很多方法可以查看这些信息，但在Linux系统和mac的OSX系统中，命令略有不同。

在mac里查看文件信息

stat -x
